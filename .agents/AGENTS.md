# Reglas del Agente para Metadea

- **Simplificación de Comentarios**: Al modificar o mejorar cualquier archivo de código, elimina los comentarios obsoletos, excesivamente detallados o basura. Deja únicamente comentarios muy concisos o pequeñas explicaciones breves si la lógica del código es extremadamente compleja. Evita explicaciones redundantes o comentarios generados por IA que dejen referencias a cosas inexistentes.
