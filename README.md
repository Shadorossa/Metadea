# Metadea

**Tu gestor personal de biblioteca multimedia todo en uno**

Metadea es una aplicación de escritorio que te permite gestionar, sincronizar y organizar toda tu colección de medios: anime, manga, videojuegos, películas, series, libros y más, todo en un solo lugar.

## 🎯 ¿Qué es Metadea?

Metadea es tu compañero perfecto si:
- Tienes una amplia colección de videojuegos en Steam, Epic Games, GOG, Xbox y EA Play
- Ves anime y quieres sincronizar tu progreso automáticamente
- Coleccionas libros, novelas visuales o cualquier tipo de media
- Quieres un único lugar para organizar toda tu biblioteca
- Deseas sincronizar tus datos con AniList, Steam y otros servicios

## ✨ Características principales

### 📚 Gestión de Biblioteca
- Organiza y categoriza toda tu colección de medios
- Listas personalizadas de favoritos
- Sistema flexible de calificación (5 estrellas, escala numérica, emojis)
- Seguimiento de progreso en series y anime

### 🎮 Gestión de Videojuegos
- Detección automática de juegos instalados en:
  - Steam
  - Epic Games
  - GOG
  - Xbox
  - EA Play
- Lanza juegos directamente desde la aplicación
- Información detallada: portadas, banners, géneros, descripción

### 📺 Anime Local
- Vincula tu carpeta de anime local
- Sincronización automática con tu lista de AniList
- Selector de episodios intuitivo
- Reproduce episodios en VLC o tu reproductor predeterminado
- Sincroniza el progreso automáticamente

### 🔗 Sincronización en Línea
- Conecta con **AniList** para sincronizar tu biblioteca de anime
- Autenticación con **GitHub**
- Integración con **Steam** para detectar juegos
- Búsqueda potente mediante **IGDB** y **TMDB**

### 🎨 Personalización
- Múltiples temas visuales
- Soporte multiidioma
- Avatar y banner personalizados
- Sistema de perfil completo

## 📥 Instalación

### Requisitos del Sistema
- **Windows**: 7 o superior (incluido Windows 11)
- **RAM**: Mínimo 2GB
- **Espacio**: ~500MB para la instalación

### Pasos de instalación

1. **Descargar el instalador**
   - Dirígete a las [Releases](https://github.com/Shadorossa/Metadea/releases)
   - Descarga el archivo `Metadea_*.msi` (la versión más reciente)

2. **Ejecutar el instalador**
   - Haz doble clic en el archivo `.msi` descargado
   - Sigue las instrucciones del asistente de instalación
   - Elige dónde instalar la aplicación (recomendado: Archivos de Programa)

3. **Aceptar permisos**
   - Windows puede pedir confirmación para instalar la aplicación
   - Haz clic en "Sí" para continuar

4. **Ejecutar la aplicación**
   - Una vez instalada, busca "Metadea" en el menú Inicio
   - O haz clic en el acceso directo en tu escritorio

## 🚀 Primeros pasos

### 1. Configurar tu perfil
- Ve a **Configuración** → **Perfil**
- Sube tu avatar y banner personalizados
- Elige tu nombre de usuario y sistema de calificación preferido

### 2. Conectar servicios (Opcional)
- **AniList**: Haz clic en "Conectar AniList" para sincronizar tu lista de anime
- **GitHub**: Conecta tu cuenta GitHub para respaldo de datos
- **Steam**: La detección de juegos es automática

### 3. Organizar tu biblioteca
- Ve a **Local** para gestionar tus medios locales
- Configura rutas para tus carpetas de anime, libros, novelas visuales, etc.

### 4. Disfrutar
- Busca medios para agregar a tu biblioteca
- Sincroniza automáticamente con AniList
- Gestiona tus listas de favoritos

## 🎮 Cómo usar

### Buscar videojuegos
1. Ve a **Local** → **Videojuegos**
2. Metadea escanea automáticamente tus plataformas de juegos
3. Haz clic en un juego para ver detalles
4. Haz clic en **Jugar** para lanzar el juego

### Ver tu anime local
1. Ve a **Local** → **Anime**
2. Selecciona tu carpeta de anime
3. Elige un anime de tu lista en AniList
4. Metadea detectará automáticamente los episodios
5. Haz clic en un episodio para reproducir en VLC
6. El progreso se sincroniza automáticamente

### Agregar favoritos
1. Busca un media (anime, manga, película, etc.)
2. Haz clic en el corazón o en "Agregar a favoritos"
3. Elige la lista de favoritos donde deseas agregarlo

### Sincronizar con AniList
1. Ve a **Configuración** → **Cuenta**
2. Haz clic en "Conectar AniList"
3. Autoriza la aplicación en AniList
4. Tus datos se sincronizarán automáticamente

## ❓ Preguntas frecuentes

**P: ¿Se sincroniza automáticamente con AniList?**
R: Sí, una vez conectes tu cuenta de AniList, todos los cambios se sincronizarán automáticamente.

**P: ¿Funciona con consolas de juegos?**
R: Actualmente detecta juegos de PC. Las consolas (PlayStation, Xbox, Nintendo) se pueden agregar manualmente.

**P: ¿Necesito estar conectado a Internet?**
R: No. Metadea funciona completamente sin Internet. La sincronización se realiza cuando estés conectado.

**P: ¿Dónde se guardan mis datos?**
R: Todo se guarda localmente en tu computadora. También puedes sincronizar con servicios en línea como AniList.

**P: ¿Puedo usar Metadea en Mac o Linux?**
R: Actualmente solo está disponible para Windows. Se están considerando versiones para otras plataformas.

**P: ¿Cómo desinstalo Metadea?**
R: Ve a **Panel de Control** → **Programas** → **Programas y características** → Busca "Metadea" → Haz clic en "Desinstalar"

## 🐛 Solución de problemas

**La aplicación no inicia**
- Intenta desinstalar y reinstalar la aplicación
- Asegúrate de tener los últimos actualizadores de Windows instalados

**VLC no abre los episodios**
- Instala [VLC Media Player](https://www.videolan.org/vlc/) si aún no lo tienes
- Configura VLC como tu reproductor predeterminado

**AniList no sincroniza**
- Ve a Configuración y reconecta tu cuenta de AniList
- Asegúrate de tener conexión a Internet

**Los juegos no aparecen**
- Asegúrate de que tus juegos están instalados en las plataformas correctas
- Intenta hacer clic en el botón de "Actualizar"
